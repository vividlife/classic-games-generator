## 功能实现

### 核心规则（lib/useGo.ts）
- **棋盘大小**：9×9 / 13×13 / 19×19 可选，默认 19×19
- **提子**：BFS 求连通块，若气数为 0 则提走（capturedByBlack / capturedByWhite 计数）
- **禁入点**：落子后若己方连通块无气（且未提对方），判定为 suicide，拒绝落子
- **打劫（Ko）**：单个提子后设置 koPoint，对方下一手不能在该点落子
- **虚手（Pass）**：连续两次虚手则对局结束
- **认输**：当前手方认输，对方获胜

### 棋盘 UI（GoBoard.tsx）
- 木纹色背景（#dcb463），棋线（#8a6225）
- 星位（9路5点，13路5点，19路9点）
- 悬停预览（半透明当前方棋子）
- 劫材禁点红框标记
- 最后落子中心点标记

### 游戏界面（GoGame.tsx）
- 开始界面：棋盘大小选择 + 规则说明
- 对局界面：双方提子数显示，当前方高亮，虚手/认输（带确认防误触）/重开按钮
- 结束界面：全屏遮罩显示结果（认输或双虚手），提子统计
