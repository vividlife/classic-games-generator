2026-03-02

## 新增文件

### `lib/useWerewolf.ts`
游戏核心逻辑 hook，包含：
- 6种角色定义（狼人、村民、预言家、女巫、猎人、守卫），含 emoji、描述、阵营
- 6人到18人的预设角色配置（PRESET_CONFIGS）
- 状态管理：setup/dealt 两阶段、玩家列表、当前展示玩家、全局解锁标志
- 操作：设置玩家数量（自动切换预设）、调整角色配置、一键恢复预设、开始发牌（Fisher-Yates 洗牌）、查看/隐藏单人身份、全局解锁/锁定、重置回设置阶段

### `components/werewolf/WerewolfDealer.tsx`
主界面组件，两阶段 UI：

**设置阶段**
- 玩家人数步进器（6-18人，±按钮）
- 角色配置表（每个角色可单独 ±调整数量，显示阵营标签）
- 实时角色总数 vs 玩家人数校验（绿色✓ / 红色✗）
- "恢复预设配置"按钮
- "开始发牌"按钮（角色总数不匹配时禁用）

**发牌阶段**
- 4列编号卡片网格
- 点击编号 → 全屏黑色遮罩展示角色（emoji + 中文名 + 描述 + 阵营标签），点击任意处关闭
- "全部解锁"按钮：上帝视角，直接在卡片上显示角色 emoji 和名称（狼人红色/好人蓝色）
- "重新发牌"按钮：返回设置阶段
- 底部显示玩家总数和狼人数量统计

### `app/werewolf/page.tsx`
页面入口，含 Header、面包屑导航（← 主页 / 🐺 狼人杀发牌）
