# 围棋游戏增加判断胜负按钮

## 需求
在围棋对局界面增加一个"判断胜负"按钮，点击后显示当前盘面如果作为终局的胜负结果。

## 任务清单

- [ ] 阅读现有围棋代码结构（lib/useGo.ts, components/go/GoGame.tsx）
- [ ] 实现数目逻辑：
  - 计算黑白双方领地（空点归属）
  - 使用简化规则：完全被一方棋子包围的空点归该方
  - 加上提子数
  - 考虑贴目（黑方贴白方3.75子，即黑方需185子胜）
- [ ] 在 GoGame.tsx 添加"判断胜负"按钮
- [ ] 点击后显示弹窗/遮罩，显示：
  - 黑方：领地 + 提子数
  - 白方：领地 + 提子数
  - 胜负结果
- [ ] 运行 `npm run build` 验证编译通过
- [ ] 将修改摘要写入 `.dispatch/tasks/add-go-score/output.md`
